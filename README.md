🏠 Turíbio Imóveis - Catálogo Imobiliário Digital

Este projeto é uma plataforma de catálogo imobiliário desenvolvida para facilitar a visualização de imóveis em Maricá e região. O sistema consiste em uma Single Page Application (SPA) moderna integrada a uma API RESTful desenvolvida em C#.
🚀 Tecnologias Utilizadas

    Front-end: HTML5, CSS3 (Custom Properties e Flexbox), JavaScript Vanilla (ES6+) e Bootstrap 5.

    Back-end: C# com ASP.NET Core (API REST).

    Identidade Visual: Design Premium "Black & Gold" com suporte a Dark Mode.

    Integrações: API do WhatsApp para geração direta de leads.

🛠️ Funcionalidades Principais

    Vitrine Interativa: Listagem dinâmica de imóveis consumindo dados via Fetch API.

    Filtros em Tempo Real: Sistema de busca por texto e faixa de preço processado no lado do cliente para maior agilidade.

    Modal de Detalhes: Visualização expandida com carrossel de fotos e descrição completa sem recarregamento de página.

    Painel Administrativo: Interface para gerenciamento de anúncios (CRUD) com upload múltiplo de imagens.

    Design Responsivo: Adaptado para dispositivos móveis e desktop, garantindo acessibilidade em qualquer tela.

📐 Arquitetura do Projeto

Como estudante de Ciência da Computação, o foco deste projeto foi aplicar conceitos de:

    Consumo de API Assíncrona: Implementação de async/await para garantir que a UI não trave durante as requisições ao servidor.

    Manipulação de DOM: Criação dinâmica de elementos baseada nos dados recebidos do back-end.

    Clean UI: Separação clara entre a lógica de negócios e a camada de apresentação.
